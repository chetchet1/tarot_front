# 260312-1 수정 진행 상황

## 작업 날짜
2026년 3월 12일

## 작업 내용

### 1. 화면 회전 방지 (세로 고정)
- **문제**: 휴대폰을 가로로 눕히면 앱 레이아웃이 깨짐
- **원인**: AndroidManifest.xml에 `screenOrientation` 속성 미설정 → 앱이 자유 회전
- **해결**: MainActivity에 `android:screenOrientation="portrait"` 추가
- **파일**: `android/app/src/main/AndroidManifest.xml`

### 2. 이벤트 페이지 헤더 겹침 수정
- **문제**: EventList의 sticky 헤더 배경이 반투명(`rgba(0,0,0,0.2)`)이라 스크롤 시 아래 콘텐츠가 비쳐 보임
- **해결**: 헤더 배경을 불투명하게 변경 (`rgba(15,10,40,0.95)`)
- **파일**: `app/views/EventList.vue`

### 3. 카드 직접뽑기 UI 개선 (분석 완료, 구현 대기)
- **문제**: 78장 카드가 240도 부채꼴로 펼쳐지는데, 좁은 세로 화면에서 카드(30x45px)가 너무 작아 탭하기 어려움
- **제안**: 좌우 스와이프 캐러셀 방식으로 변경 (한 번에 5~7장씩 표시, 중앙 카드 확대)
- **상태**: 사용자 확인 후 구현 예정

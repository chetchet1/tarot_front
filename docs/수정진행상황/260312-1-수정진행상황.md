# 260312-1 수정 진행 상황

## 작업 날짜
2026년 3월 12일

## 작업 내용

### 1. 화면 회전 방지 (세로 고정)
- **문제**: 휴대폰을 가로로 눕히면 앱 레이아웃이 깨짐
- **원인**: AndroidManifest.xml에 `screenOrientation` 속성 미설정
- **해결**: `android:screenOrientation="portrait"` 추가
- **파일**: `android/app/src/main/AndroidManifest.xml`

### 2. 이벤트 페이지 헤더 겹침 수정
- **문제**: EventList의 sticky 헤더 배경이 반투명이라 스크롤 시 콘텐츠가 비쳐 보임
- **해결**: 헤더 배경을 불투명하게 변경 (`rgba(15,10,40,0.95)`)
- **파일**: `app/views/EventList.vue`

### 3. 카드 직접뽑기 UI 개선 (부채꼴 → 6열 그리드)
- **문제**: 78장 카드가 부채꼴로 펼쳐져 좁은 화면에서 카드(30x45px)가 너무 작아 탭 불가
- **해결**: 6열 그리드 + 세로 스크롤 방식으로 변경
  - 카드 크기 확대 (aspect-ratio 2:3, max-width 60px)
  - 한 화면에 18~24장 표시, 세로 스크롤로 탐색
  - `getCardSpreadStyle()` 삼각함수 계산 제거
  - 모든 배열(3장, 캘틱크로스, 세븐스타, 컵오브릴레이션쉽)에 동일 적용
- **파일**: `app/views/CardDrawing.vue`

## 변경 파일 목록
1. `android/app/src/main/AndroidManifest.xml` — portrait 고정
2. `app/views/EventList.vue` — 헤더 배경 불투명화
3. `app/views/CardDrawing.vue` — 직접뽑기 UI 그리드 방식 변경

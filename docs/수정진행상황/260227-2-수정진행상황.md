# 260227-2 3장 배열 스크롤 불가 수정

## 작업 날짜
2026년 2월 27일

## 문제
- 3장 배열 카드를 모두 뒤집은 후 "해석 보기" 버튼이 화면 아래에 가려짐
- 스크롤이 안 되어 버튼을 누를 수 없음

## 원인
- `CardDrawing.vue`의 최상위 `.card-drawing`에 `overflow-y: hidden` 적용
- 3장 카드가 세로로 나열되면 화면 높이를 초과하지만 스크롤 불가

## 해결
- `.card-drawing`에 동적 클래스 `scrollable` 추가
- 조건: `isComplete && !hasSpecialLayout` (일반 배열 결과 화면일 때만)
- `.scrollable` 클래스: `overflow-y: auto`, `height: auto`, `min-height: 100dvh`

## 영향 범위
- **영향 받는 화면**: 3장 배열 결과 화면만
- **영향 없는 화면**: 캘틱 크로스, 세븐스타, 컵오브릴레이션십 (hasSpecialLayout=true로 제외됨)
- **영향 없는 화면**: 카드 뽑기 중 화면 (isComplete=false로 제외됨)

## 수정 파일
- `app/views/CardDrawing.vue` (template 클래스 바인딩 + CSS 추가)

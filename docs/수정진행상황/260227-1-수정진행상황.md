# 260227-1 화면 확대 시 레이아웃 깨짐 수정

## 작업 날짜
2026년 2월 27일

## 문제
- 부모님 휴대폰(Android)에서 접근성 화면 확대 설정 사용 시 카드 뽑기 화면 레이아웃이 깨짐
- "카드 뽑기" 제목 줄바꿈, 카드 이미지 비율 깨짐, 전체 레이아웃 틀어짐

## 원인
- Capacitor WebView가 Android 시스템 폰트/DPI 확대 설정을 그대로 반영
- CSS px 고정 크기 레이아웃에 시스템이 텍스트만 확대하면서 컨테이너 넘침 발생

## 해결
- `MainActivity.java`에 `setTextZoom(100)` 1줄 추가
- WebView가 시스템 폰트 확대를 무시하고 항상 100% 기준으로 렌더링

## 수정 파일
- `android/app/src/main/java/com/tarotgarden/app/MainActivity.java` (1줄 추가)

## 후속 고려사항
- 눈이 안 좋은 사용자를 위해 앱 내 자체 글꼴 크기 설정(작게/보통/크게) 추가 검토
- 기본 폰트 크기를 전체적으로 1~2px 올리는 것도 검토 가능

## 테스트 방법
1. `npm run build:cap` → `npx cap sync` → Android Studio 빌드
2. Android 설정 → 접근성 → 화면 확대 최대로 설정
3. 앱 실행하여 카드 뽑기 화면 레이아웃 정상 확인
